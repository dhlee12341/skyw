    • 4. React

  • 터미널 : node -v --> node.js 버전확인
  • 터미널 : npx create-react-app [프로젝트이름] --> react 프로젝트 생성
  • 터미널 : npm install axios --> axios 설치
  • 터미널 : npm install bootstrap react-bootstrap --> react-bootstrap 설치
  • 터미널 : npm install react-router-dom --> react-router-dom 설치
  • 터미널 : npm install web-vitals --> web-vitals 설치
  • 터미널 : npm cache clean --force --> 리액트 오류시 : 캐시초기화
  • 터미널 : netstat -ano | findstr 8080 --> 스프링부트 오류시 : 포트확인
  • 터미널 : netstat -ano | findstr 3000 --> 리액트 오류시 : 포트확인
  • 터미널 : taskkill /F /PID 프로세스ID --> 포트사용중인 경우 Kill
  • 터미널 : npm start --> React 실행
